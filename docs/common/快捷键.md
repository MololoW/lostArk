# 常用菜单快捷键

#### 地图: `M` 

#### 任务: `J`

#### 委托任务（每日委托）: `Alt + J`

#### 背包: `I`

#### 人物面板: `P`

#### 宠物功能: `Alt + P`

#### 坐骑: `Alt + V`

#### 技能面板: `K`

#### 商城: `F4`

#### 歌曲系统: `F2`

#### 动作加表情: `Y`

#### 地区完成度: `N`

#### 莫可可收集: `Alt + N`
